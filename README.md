# Sistema de Predicción Salarial con Naive Bayes

Proyecto desarrollado en Go que utiliza el algoritmo Naive Bayes para estimar el salario esperado de una persona en su primer empleo a partir de sus características académicas y profesionales.

## Características
- Implementado en Go.
- Uso de archivos CSV para almacenar datos.
- Clasificación mediante algoritmo Naive Bayes.
- Predicción del rango salarial según el perfil del usuario.

## Archivos del proyecto
- main.go: Código principal.
- datos.csv: Base de datos de entrenamiento.
- modelo.nb: Modelo generado.
- go.mod: Dependencias del proyecto.

## Ejecución

```bash
go run main.go
```
