# Predicción de Salario con Naive Bayes

Aplicación desarrollada en Go que utiliza el algoritmo Naive Bayes para estimar el salario esperado de una persona en su primer empleo.

## Características Analizadas

- Nivel académico
- Dominio del inglés
- Certificaciones
- Experiencia laboral
- Habilidades técnicas

## Algoritmo Utilizado

- Naive Bayes

## Tecnologías

- Go
- HTML
- CSS

## Objetivo

Predecir el rango salarial de un candidato a partir de sus características personales y profesionales.
